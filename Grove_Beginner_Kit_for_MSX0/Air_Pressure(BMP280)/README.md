## Grove Begginner Kit for MSX0　の温度・気圧センサー（BMP280）の動作確認用プログラムです。

センサーはMSX0のPort.A(赤)に接続します。
センサーを切り離さないで仕様するには、中央のArduinoに空のスケッチを書き込む等で、無効化する必要があります。(I2Cが正しく動作しないため)
